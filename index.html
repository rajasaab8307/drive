⚠ Cannot run as a local file

You opened this as file://... — browsers completely block all network calls from local files. Login, file loading, upload — nothing will work.

You must open this from a web server every time.

✅ Option 1 — GitHub Pages (free, permanent)
  1. Go to github.com → sign in → New repository
  2. Name it (e.g. luminary) → tick Add a README → Create
  3. Add file → Upload files → drag index.html → Commit
  4. Settings → Pages → Deploy from branch → main → Save
  5. Open https://YOUR-USERNAME.github.io/luminary/ — bookmark this URL
✅ Option 2 — Quick local server (while developing) Open terminal in the folder containing this file and run:
python -m http.server 8080
Then open http://localhost:8080 in your browser.
⚠ Must re-run every time you restart your computer.
Remember: Even on a web server, your Dropbox token expires every few hours. When the app stops working, go to dropbox.com/developers/apps → your app → Generate access token, then paste it in ⚙️ Settings.
