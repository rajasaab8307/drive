⚠ Needs a Web Server

You're opening this as a local file (file://...). Browsers block API calls from local files — this is a browser security rule that can't be worked around.

You need to host this file on a web server. The easiest free option is GitHub Pages — no coding required:

  1. Go to github.com → sign up free → click New repository
  2. Name it anything (e.g. my-files) → tick Add a README → Create repository
  3. Click Add file → Upload files → drag this index.html file → Commit changes
  4. Go to Settings → Pages → Source: Deploy from branch → main → / (root) → Save
  5. Wait ~1 min → your app is live at https://YOUR-USERNAME.github.io/my-files/
Alternative: run a local server in one command If you have Python installed, open a terminal in this file's folder and run:
python -m http.server 8080
Then open http://localhost:8080 in your browser.
